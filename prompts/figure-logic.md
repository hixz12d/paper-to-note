# 阶段 2：Figure 论证路径分析

## 任务

基于阶段 1 提取的论文结构和证据片段池，分析论文中每个 Figure 的作用，梳理作者通过 Figure 构建的论证逻辑链。

**这是整个 skill 最核心的价值区块。**

## 对每个 Figure 的分析

1. **作用**: 这个 Figure 在整篇论文的论证中扮演什么角色？（如：证明合成成功、展示性能、揭示机理、验证稳定性）
2. **内容**: 这个 Figure 具体展示了什么？（图类型、关键数据趋势、对比项）
3. **支撑结论**: 作者用这个 Figure 证明了什么具体结论？
4. **与下文关系**: 这个 Figure 的结果如何引出下一个 Figure 或下一步实验？

如果某个 Figure 有子图（a, b, c...）且各有不同作用，应分别说明。

## 整体分析

1. **整体思路**: 概括作者用 Figure 讲了一个什么故事——从什么问题出发，经过哪些验证步骤，最终得出什么结论
2. **论证逻辑链**: 用简洁的箭头链或短段落，串联所有 Figure 之间的逻辑关系

## 分析原则

- **必须分析"为什么作者要放这个图"，不能只描述"图里有什么"**
- 优先基于 Figure caption + 正文中引用该 Figure 的段落进行分析
- 图像内容作为辅助参考；当图像不可读时，完全基于文本分析并标注 [图像未参考]
- Figure 的作用不确定时明确标注 [作用待确认]，不硬编
- 如果论文中没有 Figure，输出"本文未包含 Figure，跳过此区块"，不生成空壳

## 语言规则

- 正文全部用中文撰写
- 仅以下内容保留英文原文：
  - 化合物名/分子名（如 NOD-NY, Zn-HN, TPEN）
  - 仪器/方法缩写（如 HPLC, XRD, TEM, EPR, UV-vis）
  - 细胞系名称（如 A549, Raw264.7）
- 所有可翻译的学术概念必须用中文：
  - photolysis → 光解，fluorescence → 荧光，probe → 探针
  - turn-on response → 荧光增强响应，absorption red-shift → 吸收红移
  - cation selectivity → 阳离子选择性，release → 释放
- 首次出现的重要概念可在中文后括号标注英文，之后只用中文

## 输出格式

严格按以下格式输出：

---

## Figure 论证路径

### 整体思路

（概括性段落）

### Figure 1: （简短描述）

- **作用**: ...
- **内容**: ...
- **支撑结论**: ...
- **与下文关系**: ...

### Figure 2: （简短描述）

（同结构）

### 论证逻辑链

Figure 1 (描述 → 结论)
  → Figure 2 (描述 → 结论)
  → Figure 3 (描述 → 结论)
  → ...

---

## 注意事项
- 不编造论文中不存在的信息
- 每个 Figure 的分析都要有据可依，不能空谈
